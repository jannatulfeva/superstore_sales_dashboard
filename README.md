# superstore_sales_dashboard

# Overview

- An Excel-based dashboard created to analyze sales and profit performance using structured data and visualizations

- Focuses on exploring key metrics and patterns within the dataset through charts and summary tables

- Provides a clear view of business performance using interactive elements in Excel 

# Business Objective

- Monitor overall sales and profit

- Analyze performance across different categories and segments

- Track changes in sales over time

- Present data in a clear and interactive dashboard format

# Tools & Technologies

Microsoft Excel

# Key Features

- KPI Metrics:

Total Sales

Total Profit

- Data Visualization:

Category-wise performance

Segment-wise analysis

Time-based trends (sales over period)

- Use of Excel features:

Pivot Tables

Charts

Slicers
